## 安装下载

- 下载地址 https://github.com/quick-cache/quick-cache/releases
- `npm i quick-cache-npm`
- CDN http://unpkg.com/quick-cache-npm/release/bundle.js

## 快速使用

```js
var QC = window.QuickCache;
var cache = new QC();
cache.set('a', 100);
alert(cache.get('a'))
```
